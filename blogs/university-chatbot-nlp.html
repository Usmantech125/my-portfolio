üß† Step 1: Basic Rule-Based Chatbot with NLTK

We‚Äôll define some responses and use tokenization to process user input.

import nltk
from nltk.chat.util import Chat, reflections

pairs = [
    [r"hi|hello", ["Hello! How can I help you today?"]],
    [r"what is the admission process", ["You can apply online via the university portal."]],
    [r"what is the fee structure", ["The fee varies by program. Please visit our official website."]],
    [r"who is the dean of computer science", ["Prof. Dr. Ahmad is the current dean of CS."]],
    [r"bye", ["Goodbye! Have a nice day."]]
]

chatbot = Chat(pairs, reflections)
chatbot.converse()

This will launch a command-line chatbot that responds based on pattern matching.

üåê Step 2 (Optional): Deploy with Flask

from flask import Flask, request, jsonify
import nltk
from nltk.chat.util import Chat, reflections

app = Flask(__name__)
chatbot = Chat(pairs, reflections)

@app.route("/chat", methods=["POST"])
def chat():
    user_input = request.json["message"]
    response = chatbot.respond(user_input)
    return jsonify({"response": response})

if __name__ == "__main__":
    app.run(debug=True)

This Flask API can be connected to a frontend for a complete web chatbot.
